# Profit-Sales-Analytics-Dashboard


An interactive sales analytics dashboard built using **Google Looker Studio** to visualize and analyze shoe sales performance across different states, cities, customers, and product categories.

## 🚀 Live Dashboard

🔗 **Live Dashboard:**  
https://your-looker-studio-link

## 📊 Dashboard Overview

The Wave Shoes Sales Dashboard provides a complete view of business performance through interactive visualizations and real-time insights.

### Key Metrics

- 💰 Sales Amount: 113.33M
- 💵 Cost: 71.23M
- 📦 Quantity Sold: 24.05K

- Total Sales Amount
- Total Cost

- Maximum Sales

- Interactive sales map

- City-wise sales insights

- Formal Shoes
- Semi-Casual Shoes

- Single Monk

- Loafers
- Derby Shoes
